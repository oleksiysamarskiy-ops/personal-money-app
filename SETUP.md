# Finance App

Приложение использует **IndexedDB** — встроенную БД браузера.
Никакой настройки не требуется. Данные хранятся локально в браузере.

## Запуск

```
npm install
npm run dev
```

## Деплой на Vercel

```
vercel deploy
```

Никаких переменных окружения не нужно — всё работает из коробки.
